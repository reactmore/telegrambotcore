## Usage 
Install Package:
```
composer require reactmore/telegrambotcore dev-master.
```

Copy and Create Configurations
```
 php spark botcore:publish.
```

CLI Command
```
 php spark botcore:getUpdatesCLI.

 Result : 
 **** Running BOT ****

2022-03-15 20:37:16 - Processed 2 updates

{"ok":true,"result":[]}
```
